## 2. Hardware

### 3D Printer Setup
- **Printer:** Creality Ender-3 S1 Pro
- **Enclosure:** Comgrow 3D Printer Warm Enclosure
- **Ventilation:** AC Infinity Cloudline with Controller 69
- **Filtration:** AC Infinity Carbon Filter
- **Cameras:** 
  - Logitech C920 (primary angle)
  - Arducam (secondary angle)

### Server Infrastructure
- **CPU:** Intel Xeon E5-1650 v3 @ 3.50GHz (6-core/12-thread)
- **RAM:** 128GB DDR4
- **Storage:** SSD for Docker containers and print files
- **OS:** Linux with Docker containerization

This powerful server runs our entire 3D printing stack in isolated Docker containers, providing reliable 24/7 operation with room for future expansion.
